# Notification App Frontend

This folder contains the frontend implementation for the notification system.

## Run locally

1. Change into the app folder:

```bash
cd notification_app_fe
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the URL shown in the terminal.

## Notes
- This app imports the reusable logging middleware from the repository root.
- The UI includes both an all-notifications view and a priority inbox view.
